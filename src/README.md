🌦️ Sistema de Clima em Java

Projeto desenvolvido em Java com foco no aprendizado de Programação Orientada a Objetos (POO), manipulação de listas utilizando ArrayList e implementação de operações CRUD (Create, Read, Update e Delete).

📋 Funcionalidades:

Cadastrar cidades e informações climáticas

Listar todas as cidades cadastradas

Buscar cidade pelo nome

Atualizar temperatura e condição climática

Remover cidade do sistema

Encerrar a aplicação

🛠️ Tecnologias Utilizadas:

Java

Programação Orientada a Objetos (POO)

ArrayList

Scanner para entrada de dados

Estruturas de repetição e decisão

📚 Conceitos Praticados:

Durante o desenvolvimento deste projeto foram aplicados os seguintes conceitos:

Classes e Objetos

Encapsulamento

Construtores

Getters e Setters

Listas com ArrayList

Laços de repetição (for e for-each)

Estruturas condicionais

Busca, atualização e remoção de objetos em coleções

🚀 Como Executar

Clone o repositório:
git clone https://github.com/ruanpablomota/sistema-clima-java.git
Abra o projeto em sua IDE Java preferida (IntelliJ IDEA, Eclipse ou VS Code).
Execute a classe Main.

Este projeto foi criado com fins educacionais para praticar os fundamentos da linguagem Java e consolidar conhecimentos em Programação Orientada a Objetos através da construção de um sistema CRUD em console.
